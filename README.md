# 進捗トラッカー (Progress Tracker)

動画学習やタスクの進捗を管理する PWA。総本数と締切から「1日に何本見ればよいか（今日の目標）」を自動計算します。

- テーマを枝分かれ（ツリー）構造で管理
- 締切・休みの曜日・個別休日を考慮した必要ペース／今日のノルマ表示
- 複数プロジェクト対応、同期コード／JSONでのバックアップ
- オフライン対応（Service Worker）、ホーム画面に追加してアプリとして利用可

## 技術
ビルド不要のバニラ HTML / CSS / JavaScript。静的ホスティング（GitHub Pages 等）にそのまま配置できます。

## ローカルで動かす
```sh
python3 -m http.server 8765
# http://localhost:8765 を開く
```
